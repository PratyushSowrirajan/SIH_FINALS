FACIAL EXPRESSION RECOGNITION - QUICK START
============================================

STEP 1: INSTALL
---------------
pip install scikit-learn

STEP 2: TRAIN (5 minutes)
--------------------------
python train_expressions_ml.py

Instructions:
- Press ENTER to start
- Make each expression when prompted
- Press SPACE to collect 200 samples (takes 7 seconds each)
- Expressions: HAPPY, SAD, ANGRY, SURPRISED, DISGUSTED
- Hold expression steady while collecting

STEP 3: RUN DETECTOR
--------------------
python detect_expressions_ml.py

Features:
- Detects 5 expressions in real-time
- Shows NEUTRAL when no match
- Works on everyone's face
- 60 FPS / Low latency
- Clean white mesh visualization
- Press 'Q' to exit

REQUIREMENTS
------------
- Webcam
- opencv-python>=4.8.0
- mediapipe>=0.10.5
- numpy>=1.24.0
- scikit-learn>=1.3.0

WHAT YOU'LL SEE
---------------
- Fullscreen display
- Thin white face mesh (468 landmarks)
- Expression name and confidence
- FPS and model accuracy
- Clean professional interface

TROUBLESHOOTING
---------------
Q: Model not found?
A: Run train_expressions_ml.py first

Q: Low accuracy?
A: Retrain with exaggerated expressions

Q: Slow FPS?
A: Close other applications, check webcam quality

PERFORMANCE

- Target: 30-60 FPS
- Latency: <50ms
- Model: RandomForest (200 estimators)
